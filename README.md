# 🎥 MOVIEPEDIA

React로 만든 **영화 평점 매기는 웹사이트** 🎬  
사용자가 영화 목록을 보고, 직접 평점을 남기거나 확인할 수 있는 간단한 리뷰 앱입니다.

---

## 📌 주요 기능 (Features)

- 🎞️ 영화 리스트 보기 (mock 데이터 기반)
- ⭐ 각 영화별 평점 보기 및 등록
- 💬 간단한 리뷰 목록 렌더링
- 🔄 React의 상태관리(useState)와 리스트 렌더링(map) 연습
- 🧱 컴포넌트 분리 및 재사용 구조 설계

---

## 🛠️ 사용 기술 (Tech Stack)

| 분류     | 기술                       |
